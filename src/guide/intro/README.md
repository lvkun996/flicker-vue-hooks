通过该章节，你将了解到如何快速开始使用 Flicker-Vue-Hooks

## ✨ 特性

- 🏄🏼‍♂️ 易学易用
- 🔋 支持 SSR
- 🛸 丰富的 Hooks
- 🏟️ 覆盖大部分业务场景
- 🎪 交互式 demo，身临其境
- 🎯 使用 TypeScript 构建，提供完整的类型定义文件
- 🪄 支持按需加载，减少打包体积
- 🤺 演练场，大有用武之地
- 🔐 测试完善，安全可靠

## 📦 安装

```bash
npm i flicker-vue-hooks
```

## 🔨 使用

### 引入

```typescript
import { useId } from 'flicker-vue-hooks'
```
